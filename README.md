# Key-Fx
### A script to be able to customize your keyboard by adding Sfx

## Features
* **Dynamic key detection** Unlike other soundboards, this one plays a key press sound and a key release sound for a more realistic experience (Mostly)
* Customizable soundpacks (unavailable at this moment)
* **Simple GUI** A GUI so simple, it's hard to mess things up.

## Quick start
* Open the app
* Choose a soundpack to play
* You're all set!
